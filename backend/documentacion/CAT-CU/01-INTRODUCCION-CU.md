# Introducción al catálogo de casos de uso

## Objetivo
Definir la base documental del catálogo de casos de uso del sistema como punto de organización funcional por dominio.

## Alcance
Este catálogo agrupa casos de uso por dominio y sirve como referencia para documentar comportamiento funcional de manera progresiva.

## Criterio de organización
- cada dominio mantiene su propio archivo base de casos de uso
- la estructura prioriza claridad, navegación y crecimiento incremental
- el contenido puede ampliarse sin alterar la organización general

## Nota
Este espacio está preparado para incorporar casos de uso de forma progresiva y mantenible.
